ELRC-SHARE RESTful API

ELRC-SHARE provides a RESTful API which returns Language Resource Metadata in JSON format.

API ACCESS

The API can be accessed through the SEARCH_API_URL:
https://elrc-share.eu/repository/api/v1/lr/
OR
https://elrc-share.eu/repository/api/v1/lr/search/ to enable advanced search features.

Access is granted by default to all superusers and EC members registered and logged in the ELRC-SHARE repository.

External access, for example through a cURL request, requires an API key, given either in the request url as an auth param, or in the request header:
As url param
SEARCH_API_URL/?auth=API_KEY
As header (cURL)
curl SEARCH_API_URL --header 'Authorization:API_KEY'

The API_KEY is available in the profile page of all EC members.

Haystack Filters

The API leverages the existing ELRC-SHARE repository search functionality using a condensed version the preconfigured Haystack filters.

All filters are passed as key-value pairs in the q dictionary SEARCH_API_URL/?q=key:value.

Additional filters can be added to the dictionary like: SEARCH_API_URL/?q=key1:value1&q=key2:value2.

Available Filters

Conditions of Use
Applied Filter: conditionOfUse.
Haystack filter: restrictionsOfUseFilter.
Values
  • Attribution
  • Commercial Use
  • Compensate
  • Education
  • No Derivatives
  • Non Commercial Use
  • Research
  • Share Alike
  • Other
Examples:

Get all resources restricted for non commercial use
SEARCH_API_URL/?q=conditionOfUse:Non Commercial Use

Language
Applied Filter: language.
Haystack filter: languageNameFilter.
Values
Any language name as defined by the IANA Language Subtag Registry in the Description property, is a valid value for this filter.
Examples:

Get all resources that contain English
SEARCH_API_URL/?q=language:English

Get all resources that contain English and Italian
SEARCH_API_URL/?q=language:English&q=language:Italian
